### 判断是类组件还是函数组件

函数组件如何判断用户是用来生成函数组件的还是类组件的

```js
function Component (props, context, updater) {
  this.props = props
  this.context = context
  // ...
}

// 通过prototype加一个属性，类组件是new Component生成的，可以拿到isReactComponent
// 函数组件没有new过程，拿不到isReactComponent
Component.prototype.isReactComponent = {}
```
