Amos-love-frontend v1 用的是vuepress 1，需要满足如下条件才能运行和打包

less-loader @7 // less-loader版本不要超过7

sass-loader @10 // sass-loader版本不要超过10

node v14.19.0


## deploy vuepress项目发布

使用脚本deploy.sh实现快速发布，windows对于sh的linux命令暂时有支持性问题，可用git的bash来跑脚本
```shell
bash deploy.sh // 实际运行脚本
```
