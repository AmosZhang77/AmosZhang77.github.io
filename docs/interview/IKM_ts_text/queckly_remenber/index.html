E. Compile-Time Checking（编译时检查） 这也是 TypeScript 中泛型的一个重要优势。泛型允许在编译时进行类型检查，这有助于捕捉潜在的错误，并确保代码在运行时能够按照预期工作。与 JavaScript 的动态类型系统相比，TypeScript 的编译时类型检查提供了更高的代码质量和可维护性。
